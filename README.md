# SilentMode 🤫

**AI Auto-Reply Assistant That Kills Text Anxiety**
100% Free Forever • Open Source • Privacy First

## Quick Start

1. Clone: `git clone https://github.com/silentmodeapp/silentmode.git`
2. Install: `npm install`
3. Setup `.env` file (copy from `.env.example`)
4. Run: `npx react-native run-android` or `npx react-native run-ios`
